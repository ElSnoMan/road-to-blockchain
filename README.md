## Setup

1. Clone project
2. Install packages and dependencies

   ```bash
   poetry install
   ```

3. Run the Blockchain API

   ```bash
   poetry run uvicorn blockchain:app --reload
   ```

4. Test and see the auto-generated docs by visiting `http://localhost:8000/docs`
